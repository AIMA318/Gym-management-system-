🏋️ Gym Management System

📖 Project Overview

The Gym Management System is a database-based application developed to automate and simplify the management of gym operations. The system provides a centralized platform for storing and managing information related to gym members, trainers, memberships, attendance records, workout plans, and payments. It eliminates the need for manual record-keeping and improves the efficiency, accuracy, and accessibility of gym data.

This project demonstrates the practical implementation of Database Management System (DBMS) concepts, including table creation, relationships, SQL queries, joins, views, stored procedures, and aggregate functions.

---

🎯 Objectives

- To maintain and manage gym member records efficiently.
- To store trainer information and their assignments.
- To manage membership plans and renewals.
- To track member attendance records.
- To record and monitor payment transactions.
- To provide quick access to gym-related information through a structured database.
- To reduce paperwork and improve data accuracy.

🗄️ Database Structure

The system consists of the following main tables:

Members

Stores member information such as:

- Member ID
- Name
- Age
- Gender
- Contact Information
- Membership Status

Trainers

Stores trainer details including:

- Trainer ID
- Name
- Specialization
- Contact Information

Memberships

Stores membership-related information:

- Membership ID
- Plan Type
- Start Date
- End Date
- Fee Details

Attendance

Maintains attendance records:

- Attendance ID
- Member ID
- Date
- Attendance Status

Payments

Stores payment information:

- Payment ID
- Member ID
- Amount
- Payment Date
- Payment Status
